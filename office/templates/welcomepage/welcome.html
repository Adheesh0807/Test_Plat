  • You have to reach the destination towards which the clue points.
  • On reaching the destination, the Guide with you will confirm if u have reached the correct place or not and will provide you with a coding ques.
  • You have to solve that coding question in your own laptop.
  • Code editor- VS code
  • Programming Languages allowed: PYTHON and C++

  • Once you finish the question, the Guide will run test cases on it, and if u pass, He will enter a Treasure Code which will guide u to your next clue.
